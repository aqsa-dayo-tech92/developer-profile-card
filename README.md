# 🚀 Developer Profile Card

A clean, responsive, and modern Developer Profile Card built using HTML and Tailwind CSS.

## ✨ Features
- 📱 **Fully Responsive:** Looks great on mobile, tablet, and desktop screens.
- 🎨 **Modern UI:** Clean design with smooth spacing and a professional layout.
- 🛠️ **Tech Stack Badges:** Interactive badges for React, Node.js, MongoDB, and Tailwind.

## 🛠️ Tech Stack Used
- **HTML5** - For web structure.
- **Tailwind CSS** - For rapid, utility-first styling.
